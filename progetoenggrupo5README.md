Sistema de Monitoramento Elétrico 
Este projeto simula um sistema de monitoramento para instalações elétricas, analisando medições de tensão, corrente e fator de potência.
Projeto desenvolvido para a disciplina de Computação I.

Integrantes do Grupo

Funcionalidades
Registrar medições elétricas.
Validar se os circuitos estão dentro das normas de segurança.
Gerar relatórios automáticos de não conformidade (.txt).
Módulo Extra (Opção 5): Log de Falhas e Manutenção Preventiva.

Como Rodar
1. Clone este repositório.
2. Abra o notebook no Google Colab.
3. Execute todas as células.
4. Utilize o Menu Principal para navegar entre as opções.
